# Tela de Login

3º projeto e dessa vez com mais linguagens e aprendizados!

## ❓ Objetivo:

  Utilizar HTML, JS e PHP para criar uma tela que precisa de e-mail e senha, o JavaScript e o PHP ficam responsáveis pela parte de interação e validação. HTML como de costume é o nosso "esqueleto" e o CSS utilizado para dar uma personalidade própria.
